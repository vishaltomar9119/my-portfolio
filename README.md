## Getting Started

Prerequisites
- Node.js (recommended via nvm)
- npm or yarn

Install and run locally
1. Clone the repo
   git clone https://github.com/vishaltomar9119/my-portfolio.git
2. Install dependencies
   npm install
3. Start dev server
   npm run dev
4. Build for production
   npm run build
